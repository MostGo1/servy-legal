# Servy Legal Documents

  This repository contains the official legal documents for the Servy platform.

  ## Files

  | File | Description | Version |
  |------|-------------|---------|
  | `terms.pdf` | Terms of Service (EN + SK) | v1.0 |
  | `privacy.pdf` | Privacy Policy (EN + SK, GDPR) | v1.0 |

  ## How to Update

  1. Replace the PDF file in this repo with the new version
  2. Commit with message: `legal: update terms v{X.X}` or `legal: update privacy v{X.X}`
  3. GitHub Actions will automatically:
     - Deploy the new PDFs to the live website
     - Send email notifications to all registered users

  ## Access URLs (after domain is live)
  - Terms: https://servy.app/legal/terms
  - Privacy: https://servy.app/legal/privacy
